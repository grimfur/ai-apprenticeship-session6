# AI Apprenticeship – Session 6

## Overview
This repository contains the practical work for Session 6:
- LR1.ipynb
- LR2.ipynb

## Environment
Python 3.11
pandas
numpy
matplotlib
scikit-learn

To recreate the environment:

conda create -n ai-training python=3.11
conda activate ai-training
pip install -r requirements.txt

## Structure
notebooks/  → Jupyter notebooks  
src/        → Python scripts  
data/       → datasets
